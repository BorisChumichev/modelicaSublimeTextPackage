Modelica Sublime Text Package

Install GitHub repository

Sublime Texthttp://www.sublimetext.com/ editor comes without syntax definitions for Modelicahttps://modelica.org/ language by default, so the purpose of the package is to bring Modelica language support to Sublime Text and also to provide snippets for commonly used language patterns.

Install

First Install Package Controll for Sublime Texthttps://packagecontrol.io/installation/ (if you still have not done so) and restart Sublime. Then follow these steps:

  1. Open Sublime's command palette (Cmd+Shift+P)
  2. Enter: Package Control: Install Package
  3. Enter: Modelica

Syntax definitions will be applyed for all sources with 'mo' extention. Or you could manually enable syntax through command palette: 'Set Syntax: Modelica'.

Contribute

Everyone can contribute to the package development. Package sources are hosted at GitHubhttps://github.com/BorisChumichev/modelicaSublimeTextPackage, you are welcome to submit a pull request or create an issue.
