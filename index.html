Sublime Texthttp://www.sublimetext.com/ editor comes without syntax definitions for Modelicahttps://modelica.org/ language by default, so the purpose of this package is to bring Modelica language support to Sublime Text. In addition it provides some handy snippets for commonly used language patterns.

Also, this package is used by github/linguist https://github.com/github/linguist library in order to perform syntax highlighting of Moldelica sources at github.com.

Install

First install Package Control for Sublime Texthttps://packagecontrol.io/installation/ (if you still have not done so) and restart Sublime Text. Then follow these steps:

  1. Open Sublime's command palette (Cmd + Shift + P)
  2. Enter: “Package Control: Install Package”
  3. Enter: “Modelica”

Syntax definitions will be applyed for all sources with “mo” extention. Or you could manually enable syntax through command palette: “Set Syntax: Modelica”.

Atom.io editor support

Thanks to Matthis Thoradehttps://github.com/thorade there is an Atom.io editor cenversion of this package. For more details on how to use it check it's repository.

Contribute

Everyone can contribute to the package development. Package is not well tested yet so feedback from first users is really appreciated. Please create an issue https://github.com/BorisChumichev/modelicaSublimeTextPackage/issues if you find any bugs.
