A Sublime Text Package for the Modelica Language. 

Modelica: http://www.modelica.org

The Package is currently under construction. 

Installation:

0a. Install Package Controll for Sublime Text: http://wbond.net/sublime_packages/package_control/installation
0b. Restart Sublime Text 

1. Ctrl+Shift+P -> Add Repository
2. Enter: https://github.com/broyn/modelicaSublimeTextPackage.git 
3. Ctrl+Shift+P -> Install Package -> search for "modelicaSublimeTextPackage"
